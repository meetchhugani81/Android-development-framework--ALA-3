# 💰 Expense Tracker App - ALA 3

## 🚀 Overview

This project is a simple Flutter-based Expense Tracker application that allows users to record and manage their daily expenses.

Users can enter a title and amount for each expense, making it easy to track spending in a minimal and user-friendly interface.

---

## 🛠️ Tech Stack

* Flutter
* Dart
* (Optional: Local Storage if implemented)

---

## ✨ Features

* Add expenses with title and amount
* Simple and clean UI
* Lightweight and easy to use
* Real-time expense entry

## ⚙️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/meetchhugani81/Android-development-framework--ALA-3.git
```

2. Navigate to project folder:

```bash
cd Android-development-framework--ALA-3
```

3. Install dependencies:

```bash
flutter pub get
```

4. Run the app:

```bash
flutter run
```
